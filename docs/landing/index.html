Tudo que sua competição precisa

Projetado para competições reais de Powerlifting seguindo as regras da IPF.

⚡

Tempo Real

Sincronização instantânea via WebSocket entre todos os dispositivos conectados. Sem delay, sem refresh.

🏴

Cartões IPF

Suporte completo a cartões de penalidade IPF: amarelo, vermelho e vermelho+amarelo. Exibidos automaticamente na tela de display.

⏲

Cronômetro e Intervalos

Timer oficial com countdown, alertas sonoros nos segundos finais e gerenciamento completo de intervalos entre tentativas.

📱

PWA Multi-Dispositivo

Progressive Web App que funciona em celular, tablet, notebook e TV. Instale na tela inicial para acesso rápido.

🎥

Chroma Key

Tela com fundo verde para captura no OBS Studio. Sobreponha as luzes dos árbitros na sua transmissão ao vivo.

🔒

Sessões com PIN

Cada sessão é protegida por PIN. QR Codes individuais para cada árbitro com tokens JWT rotativos.

🌐

3 Idiomas

Interface disponível em Português, Inglês e Espanhol. Detecção automática do idioma do navegador.

📊

Analytics

Dashboard master com métricas de uso, sessões ativas, distribuição geográfica e histórico de decisões.

🖥

Windows Portátil

Pacote Windows com Node.js embutido. Basta extrair o ZIP e executar — sem instalação necessária.

Perguntas Frequentes

É realmente gratuito?
Sim! Referee Lights é 100% gratuito e open-source sob licença MIT. Pode usar em qualquer competição sem custo.
Funciona no celular?
Sim. É um PWA que funciona em qualquer dispositivo com navegador. Árbitros podem usar celular, tablet ou notebook. Pode ser instalado na tela inicial.
Posso usar para transmissão ao vivo?
Sim! A tela de chroma key pode ser capturada no OBS Studio ou qualquer software de streaming. Sobreponha as decisões dos árbitros na sua live.
Preciso de internet?
Não necessariamente. Para uso local, basta uma rede WiFi entre os dispositivos. O servidor roda na sua máquina e todos se conectam via rede local.
Segue as regras da IPF?
Sim. O sistema implementa o fluxo completo de decisão com 3 árbitros, cartões de penalidade (amarelo, vermelho, vermelho+amarelo) e temporização conforme regras IPF.
Como configuro para minha competição?
Basta instalar via uma das opções de deploy (local, Docker ou Windows portátil), criar uma sessão no painel admin e compartilhar os QR codes com os árbitros. Leva menos de 5 minutos.
